Lomeye Icon

Version 1.01 ,508+ CSS Icons

最适合中文开发环境的、基于CDN的纯CSS免费图标，全面兼容IE6+、Chrome、Safari、Firefox等浏览器，不需要下载安装即可在任意页面随意调用。

  • 下载zip压缩档
  • 下载WordPress插件

扫一扫，分享给小伙伴儿！

Step 1 : 安装

方式 1 CDN调用

在<head>中引用以下代码：


            <!-- Lomeye Icon 核心 CSS 文件 -->
            <link rel="stylesheet" href="http://lomeye-icon.qiniudn.com/lomeye-icon-cdn.css" />
            <!-- Lomeye Icon 兼容IE6-7 --> 
            <!--[if lt IE 8]>
	            <link rel="stylesheet" href="http://lomeye-icon.qiniudn.com/ie7.css" />
	            <script src="http://lomeye-icon.qiniudn.com/ie7.js"></script>
              <![endif]-->

方式 2 下载至本地

下载压缩包之后，将其解压缩到任意目录即可看到以下目录结构：


       lomeye-icon/
            ├── lomeye-icon.css
            ├── ie7/
            │   ├── ie7.css
            │   ├── ie7.js
            └── fonts/
                ├── lomeye-icon.eot
                ├── lomeye-icon.svg
                ├── lomeye-icon.ttf
                └── lomeye-icon.woff

复制除demo以外的文件到你需要的地方，并引用到<head>中：


            <!-- Lomeye Icon 核心 CSS 文件 -->
            <link rel="stylesheet" href="lomeye-icon/lomeye-icon.css">
            <!-- Lomeye Icon 兼容IE6-7 --> 
            <!--[if lt IE 8]>
	            <link rel="stylesheet" href="lomeye-icon/ie7/ie7.css" />
	            <script src="lomeye-icon/ie7/ie7.js"></script>
              <![endif]-->

方式 3 在<body>中引用

如果你是在单独的BLOG文章或者别的地方（只要文章主体可以编辑html代码），不能修改<head>代码的时候，你可以直接在<body>中使用图标之前引用样式文件：


          <body>
            ...
            <style type="text/css">
              @import url("../lomeye-icon/lomeye-icon.css"); /*本地调用*/
              @import url("http://lomeye-icon.qiniudn.com/lomeye-icon-cdn.css"); /*或者从CDN调用，特别推荐，即使不能安装都可以使用*/
            </style>
            ...
          </body>

特别提示：如果使用CDN调用地址，你甚至不用对现有页面进行任何修改就能使用Lomeye Icon的图标。

Step 2 : 使用

使用<span>标签调用CSS样式：


            <span class="ico-lomeye"></span>

显示结果： Lomeye网站图标（鼠标划过放大效果）

Step 3 : 调整

我们看到，上面的图标跟页面中body的文字尺寸是相同的，但是太小了，我们可以使用.ico-lg .ico-xl .ico-2x .ico-3x .ico-4x来控制图标的输出尺寸： 


            <span class="ico-lomeye ico-lg"></span><span> L号图标</span>
            <span class="ico-lomeye ico-xl"></span><span> XL号图标</span>
            <span class="ico-lomeye ico-2x"></span><span> 2XL号图标</span>
            <span class="ico-lomeye ico-3x"></span><span> 3XL号图标</span>
            <span class="ico-lomeye ico-4x"></span><span> 4XL号图标</span>

显示结果：

L号图标 XL号图标 2XL号图标 3XL号图标 4XL号图标

但是.ico-lg .ico-xl .ico-2x .ico-3x .ico-4x不能用于图标的外层容器，否则后面的文字也会相应增加字号：


            <div class="ico-2x"><span class="ico-lomeye"></span><span> 2XL号图标</span></div>

则会输出成

错误： 2XL号图标

不过你可以使用.ico-lg-in .ico-xl-in .ico-2x-in .ico-3x-in .ico-4x-in来代替它：


            <div class="ico-2x-in"><span class="ico-lomeye"></span><span> 2XL号图标</span></div>

正确： 2XL号图标

关于小号图标：

因为图标的尺寸是基于父级容器所设定的font-size属性，不同大小的图标都是用em为单位做的增倍，要获得小号图标，你只需要在图标外层添加<small>...</small>标签就可以了。当然，它也能支持所有的html文本格式标签。
